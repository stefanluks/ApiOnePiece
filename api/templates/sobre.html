Sobre nós

O que é este projeto?

Este projeto é uma API REST de One Piece construida com Django Rest Framework. Usando o sistema você tem acesso as informações dos milhares de personagens existentes e catalogados na obra.
Comece lendo a Documentação

Quem somos nós?

Sou Stefan Lucas, um programador apaixonado por One Piece.

Por quê construir isso?

Durante a graduação sempre que aparecia atividade de implemtação que necessitava de fazer uso de alguma API eu ficava me perguntando por quê não existia uma de One Piece. O anime é muito rico em informações e com um universo tão grande, seria uma otima base de informações e bem grande.
